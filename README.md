# rx-gimme

It's a graphical python tool you can use to get all links on a webpage.

Requires: beautifulsoup4 (4.8.1)

Developed in python 3.7.5.

Tested under Windows 10, MacOS Catalina and Linux

## Features
- Returns on special buffer
- Buffer can be copied or saved directly
- Completes relative links with the base url
